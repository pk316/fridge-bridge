# Fridge Bridge

### Technologies: JavaScript, jQuery, Bootstrap, Ajax, CSS3, HTML5

----

Fridge Bridge is a food and drink recipe application which incorporates the Youtube, Yummly, and CocktailDB APIs

![Fridge-Bridge](images/readme.png)
